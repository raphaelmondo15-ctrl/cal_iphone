# IPHONE CALCULATOR JS

A simple, secure, and responsive web-based calculator built using HTML, CSS, and JavaScript.
This project uses mathjs to safely evaluate mathematical expression instead of using eval().

## Feature

The calculator can procce mathemtical expression (+ - x /).
It also evaluate advanced math functions.
Percentage support.
Decimal number.
With no eval().

### Security

This project does not use eval() or the Function contructor.
validate input.
Parse numbers properly

HTML for structure
